API Project: Request Header Parser Microservice

    User Stories :

  1. The API endpoint is GET https://inconclusive-hydrofoil.glitch.me/api/whoami
  2. The API gets the IP address, preferred languages (from headers.accept-language)
    and system information (from headers.user-agent) for a device.

Example Usage:

https://inconclusive-hydrofoil.glitch.me/api/whoami

Example Output:

{"ipaddress":"::ffff:159.20.14.100","language":"en-US,en;q=0.5",
"software":"Mozilla/5.0 (X11; Ubuntu; Linux x86_64; rv:50.0) Gecko/20100101 Firefox/50.0"}
